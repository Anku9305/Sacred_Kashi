# Sacred Kashi — Website

Authentic Vedic Rituals & Sacred Ceremonies website for Varanasi, built for international tourists.

## 🚀 GitHub Pages par FREE deploy kaise karein (Step-by-step)

### Step 1: GitHub par naya repository banayein
1. [github.com](https://github.com) par jaayein aur login karein (account nahi hai to free signup karein)
2. Top-right corner mein **"+"** icon par click karein → **"New repository"**
3. Repository ka naam likhein, jaise: `sacred-kashi` (ya kuch bhi)
4. **Public** select karein (zaroori hai GitHub Pages ke liye)
5. **"Create repository"** par click karein

### Step 2: Files upload karein
1. Apne naye repository page par, **"uploading an existing file"** link par click karein
2. Is folder ki teen files ko drag-and-drop karein:
   - `index.html`
   - `style.css`
   - `script.js`
3. Neeche **"Commit changes"** button par click karein

### Step 3: GitHub Pages on karein
1. Repository ke top mein **"Settings"** tab par jaayein
2. Left sidebar mein **"Pages"** par click karein
3. **"Branch"** ke section mein, `main` branch select karein aur `/ (root)` folder rakhein
4. **"Save"** par click karein
5. 1-2 minute wait karein, phir page reload karein

### Step 4: Aapki website live ho gayi! 🎉
Aapko ek link milega is tarah ka:
```
https://your-username.github.io/sacred-kashi/
```
Yahi link aap WhatsApp, Instagram bio, Google Business profile — har jagah share kar sakte hain.

## ✏️ Website mein khud edit kaise karein

Saari files plain HTML/CSS/JS hain — koi technical software install karne ki zaroorat nahi.

- **Text/content badalne ke liye**: `index.html` file ko GitHub par hi edit kar sakte hain (pencil icon par click karke)
- **Colors/design badalne ke liye**: `style.css` ke top mein `:root { }` section mein saare colors hain
- **WhatsApp number badalne ke liye**: `index.html` mein `919305433808` ko dhundh kar apna number daal dein (sab jagah ek saath)
- **Instagram/Facebook link daalne ke liye**: `index.html` mein Footer section mein "Follow" ke neeche `href="#"` ko apne actual links se replace karein

Agar future mein `sacredkashi.com` jaisa apna domain chahte hain (₹500-1000/year), to:
